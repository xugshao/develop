Git基本常用命令如下：


mkdir        XX (创建一个空目录 XX指目录名)


pwd          显示当前目录的路径。


git init          把当前的目录变成可以管理的git仓库，生成隐藏.git文件。


git add XX       把xx文件添加到暂存区去。


git commit Cm “XX”  提交文件 Cm 后面的是注释。


git status        查看仓库状态


git diff XX      查看XX文件修改了那些内容


git log          查看历史记录
git reset --hard HEAD^ 或者 git reset --hard HEAD~ 回退到上一个版本

(如果想回退到100个版本，使用git reset Chard HEAD~100 )


cat XX         查看XX文件内容


git reflog       查看历史记录的版本号id
git checkout -- XX  把XX文件在工作区的修改全部撤销。


git rm XX          删除XX文件


git remote add origin https://github.com/18515592159zhu/testgit.git 关联一个远程库


git push Cu(第一次要用-u 以后不需要) origin master 把当前master分支推送到远程库


git clone https://github.com/18515592159zhu/testgit.git  从远程库中克隆


git checkout Cb dev  创建dev分支 并切换到dev分支上


git branch  查看当前所有的分支


git checkout master 切换回master分支


git merge dev    在当前的分支上合并dev分支


git branch Cd dev 删除dev分支
git branch name  创建分支


git stash 把当前的工作隐藏起来 等以后恢复现场后继续工作


git stash list 查看所有被隐藏的文件列表


git stash apply 恢复被隐藏的文件，但是内容不删除


git stash drop 删除文件


git stash pop 恢复文件的同时 也删除文件


git remote 查看远程库的信息


git remote Cv 查看远程库的详细信息


git push origin master  Git会把master分支推送到远程库对应的远程分支上



1、pwd  显示当前的目录

2、git init  把这个目录变成git可以管理仓库

3、git add <file>  添加文件到暂存区

4、git commit -m '提交说明'  把文件提交到仓库

5、git status  查看是否还有文件未提交

6、git diff <file>  查看文件改了什么内容

7、git log  查看历史记录
git log -pretty=oneline 简略显示

8、git reset -hard HEAD^ 回退版本 HEAD^^表示回退到上上个版本
gif reset -hard HEAD~100

9、cat <file>  查看文件内容

10、git reflog  查看版本号

11、git reset -hard +版本号  按版本号回退到指定的版本

12、git checkout  ― file  可以丢弃工作区的修改

13、rm <file>  删掉文件


14、ssh-keygen -t rsa -C "账号" 建立本地git仓库与github仓库的传输  id_rsa,id_rsa.pub
ssh keys >> add ssh key >> id_rsa.pub内容 >>add key


把一个已有的本地仓库与之关联，然后，把本地仓库的内容推送到GitHub仓库

15、git remote add origin github仓库地址  
    
    要查看远程库的信息 使用 git remote
    
    要查看远程库的详细信息 使用 git remote Cv

16、git push -u origin master  把本地的master分支内容推送的远程新的master分支，还会把本地的master分支和远程的master分支关联起来 要输入github的用户名和密码

17、git push origin master  把本地master分支的最新修改推送到github上

18、git clone github仓库地址  克隆一个本地库

19、git branch name 创建分支

    git branch  查看分支

    git checkout name  切换分支

    git checkout -b name  创建并切换分支

    git merge name  合并某分支到当前分支

    git branch -d name  删除分支



20、git stash  隐藏当前分支
